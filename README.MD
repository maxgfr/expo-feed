# expo-feed

Application which displays social media contents stemming from :
- Youtube
- Twitch
- Spotify
- Instagram

It uses embed functionality combined with `WebView` from `react-native`

## Run the project

```
    git clone https://github.com/maxgfr/expo-feed.git
    cd expo-feed
    npm install
    expo start
```
